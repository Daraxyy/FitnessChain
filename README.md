# FitnessChain

A decentralized fitness activity tracking and reward system for incentivizing healthy lifestyle choices on Stacks blockchain.

## Features

- Workout session tracking with intensity-based rewards
- Athlete level progression system with bonus multipliers
- Fitness point accumulation and redemption
- Training plan optimization with time-based penalties
- Comprehensive fitness program statistics

## Smart Contract Functions

### Public Functions
- `start-workout-session` - Begin fitness workout session
- `complete-workout-session` - Complete workout and earn rewards
- `claim-fitness-rewards` - Claim accumulated fitness points
- `optimize-training-plan` - Optimize training with point investment
- `complete-training-optimization` - Complete optimization process

### Read-Only Functions
- `get-workout-count` - Get user's total workout sessions
- `get-fitness-point-balance` - Get user's fitness point balance
- `get-athlete-level` - Get user's athlete level
- `get-fitness-program-stats` - Get overall program statistics

## Reward System
- Base reward: 20 points per workout
- Athlete bonus: 7 points per level (max level 10)
- Optimization multiplier: 3x for training plans

## Usage

Deploy the contract to create a fitness incentive system where users can track workouts, earn rewards, and optimize their training plans.
